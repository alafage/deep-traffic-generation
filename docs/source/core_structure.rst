Core Structure
==============

The `deep_traffic_generation` library provides the following classes:

- PyTorch Modules

	- `FCN <core/fcn.html>`_ (Fully-Connected Network);
	- `RNN <core/rnn.html>`_ (Recurrent Neural Network);
	- `TCN <core/tcn.html>`_ (Temporal Convolution Network);

- PyTorch Lightning Modules:

	- `AE <core/ae.html>`_ (Autoencoder);
	- `VAE <core/vae.html>`_ (Variational Autoencoder);

- PyTorch Datasets:
  
	- `TrafficDataset <core/traffic_dataset.html>`_;
